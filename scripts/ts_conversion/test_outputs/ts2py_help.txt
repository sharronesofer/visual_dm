usage: ts2py.py [-h] (--input INPUT | --batch BATCH) [--output OUTPUT]
                [--output-dir OUTPUT_DIR]

Convert TypeScript to Python

options:
  -h, --help            show this help message and exit
  --input INPUT         TypeScript file to convert
  --batch BATCH         Path to file containing list of TypeScript files to
                        convert
  --output OUTPUT       Output Python file path
  --output-dir OUTPUT_DIR
                        Output directory for batch conversion
